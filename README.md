**Votação API**

====================

#### Tecnologias Utilizadas

* Java 8
* Maven 3.1.1
* Redis 3.2.9
* MongoDB 3.4.4

====================

 #### Para rodar a aplicação
 
 ##### `mvn clean install` na raiz da aplicação.
 
 Depois rode a aplicação:
 
 ##### ` java -jar target/votacao-api.jar` na raiz da aplicação.
